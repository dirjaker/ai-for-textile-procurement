## 📖 项目简介

本项目旨在帮助棉料编织行业的坯布采购经理快速掌握：
- AI 基础常识与核心概念（面向非技术人员）
- Codex 等 AI Agent 工具的使用方法
- AI 在采购日常工作中的实际应用场景
- 采购专业知识与面试准备

**特色：**
- ✅ 零基础入门，通俗易懂
- ✅ 理论结合实践，学以致用
- ✅ 丰富的面试题和参考答案
- ✅ Codex 完整教程（含国产模型配置）

## 📚 内容概览

### 第一部分：知识文档（docs/）

#### 第一章：什么是 AI？
- AI 的定义和核心概念
- AI 的发展历程
- 机器学习和深度学习基础
- AI 能做什么、不能做什么
- 常见误解澄清

#### 第二章：什么是大语言模型（LLM）？
- LLM 的定义和工作原理
- 主流大模型对比
  - 国际：GPT-4、Claude 3、Gemini、Llama 3
  - 国产：DeepSeek、通义千问、文心一言、智谱清言、讯飞星火
- 模型能力与局限性
- 如何选择适合的模型

#### 第三章：什么是 AI Agent？
- Agent 的定义和核心组件
- Agent 与传统 AI 的区别
- Agent 工作流程
- 主流 Agent 工具：Codex、Claude Code、Cursor、GitHub Copilot
- 采购场景应用

#### 第四章：AI 能力边界
- AI 非常擅长的事（数据处理、文本生成、信息提取）
- AI 需谨慎的事（数值计算、专业判断、预测分析）
- AI 不适合的事（创造性、情感理解、实时信息）
- 采购场景的 AI 应用指南

#### 第五章：Codex 使用教程
- Codex 简介和安装准备
- 安装步骤详解
- 基本使用方法
- **连接国产模型（cc-switch 教程）**
  - DeepSeek 配置
  - 通义千问配置
  - 智谱 GLM 配置
  - 文心一言配置
- 采购实战案例
