# AI for Textile Procurement

> 专为棉料编织行业坯布采购经理设计的 AI 学习指南

[![GitHub](https://img.shields.io/badge/GitHub-dirjaker-181717?style=flat&logo=github)](https://github.com/dirjaker)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## 📖 项目简介

本项目旨在帮助棉料编织行业的坯布采购经理快速掌握：
- AI 基础常识与核心概念
- Codex 等 AI Agent 工具的使用方法
- AI 在采购日常工作中的实际应用场景

## 📁 项目结构

```
ai-for-textile-procurement/
├── README.md                          # 项目说明
├── 01-ai-basics-and-agent.md          # 第一部分：AI 常识与 Agent 知识总结
├── 02-procurement-ai-interview.md     # 第二部分：采购行业 AI 面试题
└── LICENSE                            # MIT 许可证
```

## 📚 内容概览

### 第一部分：AI 常识与 Agent 知识总结

- **AI 基础概念**：机器学习、深度学习、大语言模型
- **AI Agent 详解**：什么是 Agent、Codex/Cursor/Claude 等工具对比
- **Prompt Engineering**：如何与 AI 高效沟通
- **AI 在采购中的应用**：供应商评估、价格预测、合同分析等
- **实用工具推荐**：适合采购工作的 AI 工具清单

### 第二部分：面试题精选

- **采购专业知识**：坯布采购流程、供应商管理、质量控制
- **AI 应用场景**：如何用 AI 提升采购效率
- **综合能力**：问题解决、沟通协作、创新思维

## 🎯 适用人群

- 棉料编织行业坯布采购经理
- 纺织行业从业者想了解 AI 应用
- 准备采购岗位面试的求职者

## 📝 如何使用

1. 阅读 `01-ai-basics-and-agent.md` 学习 AI 基础知识
2. 阅读 `02-procurement-ai-interview.md` 准备面试
3. 结合实际工作场景实践 AI 工具
