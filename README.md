## 🎯 这是什么？

## 🚀 快速开始

```bash
git clone https://github.com/dirjaker/ai-for-textile-procurement.git
cd ai-for-textile-procurement
npm install
npm run dev    # 本地预览 http://localhost:5173
npm run build  # 构建静态站点
```
