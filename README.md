## 📖 项目简介

本项目旨在帮助棉料编织行业的坯布采购经理快速掌握：
- AI 基础常识与核心概念（面向非技术人员）
- Codex 等 AI Agent 工具的使用方法
- AI 在采购日常工作中的实际应用场景
- 采购专业知识与面试准备

**特色：**
- ✅ 零基础入门，通俗易懂
- ✅ 理论结合实践，学以致用
- ✅ **30+ 个 AI 发展里程碑**，每个都有采购启示
- ✅ **6 个采购实际案例**，完整的 Prompt 和操作流程
- ✅ **Codex 插件与 Skill 系统**，从入门到进阶
- ✅ **深度追问 + 真实面试真题**，贴近真实面试场景
- ✅ Codex 完整教程（含国产模型配置）

## 📚 内容概览

### 第一部分：知识文档（docs/）

#### 第一章：什么是 AI？
- AI 的定义和核心概念
- **AI 发展历程（30+ 里程碑）**
  - 早期探索阶段（1950-1990）
  - 机器学习崛起（1997-2015）
  - 大模型时代（2017-至今）
  - **每个里程碑对采购的启示**
- 机器学习和深度学习基础
- AI 能做什么、不能做什么
- 常见误解澄清

#### 第二章：什么是大语言模型（LLM）？
- LLM 的定义和工作原理
- 主流大模型对比
  - 国际：GPT-4、Claude 3、Gemini、Llama 3
  - 国产：DeepSeek、通义千问、文心一言、智谱清言、讯飞星火
- 模型能力与局限性
- 如何选择适合的模型
- **⭐ 采购工作中的 6 个实际案例（重点！）**
  - 紧急询价处理（30分钟 vs 传统3-4小时）
  - 供应商年度评估（3.5小时 vs 传统1周）
  - 采购数据分析（1小时 vs 传统2天）
  - 合同条款审查（30分钟 vs 传统4小时）
  - 市场行情跟踪（25分钟 vs 传统3小时）
  - 采购谈判准备（50分钟）

#### 第三章：什么是 AI Agent？
- Agent 的定义和核心组件
- Agent 与传统 AI 的区别
- Agent 工作流程
- 主流 Agent 工具：Codex、Claude Code、Cursor、GitHub Copilot
- 采购场景应用

#### 第四章：AI 能力边界
- AI 非常擅长的事（数据处理、文本生成、信息提取）
- AI 需谨慎的事（数值计算、专业判断、预测分析）
- AI 不适合的事（创造性、情感理解、实时信息）
- 采购场景的 AI 应用指南

#### 第五章：Codex 使用教程
- Codex 简介和安装准备
- 安装步骤详解
- 基本使用方法
- **连接国产模型（cc-switch 教程）**
  - DeepSeek 配置
  - 通义千问配置
  - 智谱 GLM 配置
  - 文心一言配置
- 采购实战案例
- **⭐ Codex 插件与 Skill 系统（新增！）**
  - Skill 概念和使用方法
  - 常用 Skill 推荐（9个）
  - 插件系统（5个推荐插件）
  - 推荐的 Skill 组合（3种场景）
  - 构建自己的工具链

3. 阅读 docs/03-什么是AI-Agent.md（40 分钟）
4. 阅读 docs/04-AI能力边界.md（30 分钟）
```

### 路径二：学习 Codex 使用

```
1. 阅读 docs/05-Codex使用教程.md
2. 按照教程安装 Codex

4. 尝试采购实战案例
