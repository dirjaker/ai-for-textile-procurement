# 情景模拟题

> 真实场景下的综合能力考核

---

## 一、采购决策类

### 情景 1：紧急采购决策

**背景：**
生产线急需一批坯布，但你的常规供应商无法在要求的时间内交货。你找到一家新供应商，价格比常规供应商高 15%，但可以按时交货。你会怎么做？

**考察点：** 紧急决策能力、成本意识、风险评估

**评分维度：**
- 问题分析能力（20%）
- 解决方案多样性（25%）
- 成本控制意识（20%）
- 风险评估（20%）
- 沟通协调（15%）

**参考答案思路：**
1. 评估紧急程度和影响
2. 考虑多种解决方案：
   - 接受高价，确保生产
   - 寻找其他供应商
   - 与常规供应商协商加急
   - 调整生产计划
3. 评估各方案的利弊
4. 做出决策并沟通

---

### 情景 2：供应商涨价

**背景：**
你的主要供应商突然通知涨价 20%，理由是原材料成本上升。公司要求你控制成本，不允许涨价。你会如何处理？

**考察点：** 谈判能力、问题解决、成本分析

**评分维度：**
- 原因分析（20%）
- 谈判策略（25%）
- 替代方案（25%）
- 沟通技巧（15%）
- 长期规划（15%）

**参考答案思路：**
1. 验证涨价原因：
   - 查询原材料市场价格
   - 了解行业整体情况
2. 准备谈判：
   - 分析供应商成本结构
   - 准备替代供应商信息
   - 明确谈判底线
3. 谈判策略：
   - 长期合作换取价格稳定
   - 增加采购量换取折扣
   - 调整付款条件
   - 分阶段涨价
4. 备选方案：
   - 开发新供应商
   - 使用替代材料
   - 与领导沟通调整目标

---

### 情景 3：质量问题处理

**背景：**
你采购的一批坯布在后加工时发现质量问题，导致客户投诉。这批布已经使用了一部分。你会如何处理？

**考察点：** 问题处理、供应商管理、客户沟通

**评分维度：**
- 应急处理（25%）
- 原因追溯（20%）
- 责任界定（20%）
- 解决方案（20%）
- 预防措施（15%）

**参考答案思路：**
1. 立即行动：
   - 停止使用问题布料
   - 隔离问题批次
   - 评估损失范围
2. 原因调查：
   - 调取检验记录
   - 与供应商沟通
   - 第三方检测
3. 责任界定：
   - 根据合同条款
   - 检验报告
   - 双方协商
4. 解决方案：
   - 退货/换货
   - 索赔损失
   - 客户沟通和补偿
5. 预防措施：
   - 加强检验
   - 完善合同条款
   - 供应商评估调整

## 二、供应商管理类

### 情景 4：供应商开发

**背景：**
公司要求你开发 3 家新的坯布供应商，以降低供应链风险。你只有 2 个月时间。你会怎么做？

**考察点：** 供应商开发能力、时间管理、评估能力

**评分维度：**
- 信息收集（20%）
- 筛选标准（20%）
- 评估方法（25%）
- 时间规划（20%）
- 沟通能力（15%）

**参考答案思路：**
1. 信息收集：
   - 行业展会
   - 同行推荐
   - 网络搜索
   - 行业协会
2. 初步筛选：
   - 产品范围匹配
   - 资质齐全
   - 规模合适
3. 深入评估：
   - 样品测试
   - 现场考察
   - 价格谈判
   - 背景调查
4. 时间规划：
   - 第 1-2 周：信息收集
   - 第 3-4 周：初步筛选
   - 第 5-6 周：深入评估
   - 第 7-8 周：试用和确认

---

### 情景 5：供应商关系维护

**背景：**
你的一家战略供应商因为公司内部调整，对接的业务员离职了。新业务员对你的业务不熟悉，服务明显下降。你会怎么做？

**考察点：** 关系管理、沟通能力、问题解决

**评分维度：**
- 问题识别（15%）
- 沟通策略（30%）
- 解决方案（25%）
- 长期维护（20%）
- 备用计划（10%）

**参考答案思路：**
1. 主动沟通：
   - 拜访供应商，认识新业务员
   - 介绍你的业务特点和需求
   - 建立个人关系
2. 信息传递：
   - 整理历史合作资料
   - 明确质量标准和要求
   - 约定沟通机制
3. 高层沟通：
   - 与供应商管理层沟通
   - 强调合作关系的重要性
   - 要求配备经验丰富的对接人
4. 备用方案：
   - 与备用供应商保持联系
   - 评估是否需要调整份额

## 三、成本控制类

### 情景 6：年度降本目标

**背景：**
公司给你下达了年度降本 8% 的目标，但目前市场价格在上涨。你会如何完成这个目标？

**考察点：** 成本分析、创新思维、执行力

**评分维度：**
- 成本分析（25%）
- 降本策略（30%）
- 可行性评估（20%）
- 执行计划（15%）
- 效果预测（10%）

**参考答案思路：**
1. 成本分析：
   - 分析成本构成
   - 识别降本空间
   - 评估市场价格趋势
2. 降本策略：
   - 集中采购：提高议价能力
   - 供应商整合：减少数量，增加份额
   - 付款优化：利用账期降低资金成本
   - 规格优化：避免过度规格化
   - 替代材料：使用性价比更高的材料
   - 流程优化：减少浪费和损耗
3. 执行计划：
   - 分解目标到季度
   - 制定具体行动项
   - 定期跟踪进度

### 情景 7：库存积压处理

**背景：**
你发现仓库里积压了价值 50 万元的坯布，已经存放了 6 个月。这些布料是按客户订单采购的，但客户订单取消了。你会怎么处理？

**考察点：** 问题处理、资源利用、沟通能力

**评分维度：**
- 问题分析（20%）
- 解决方案（35%）
- 损失控制（20%）
- 沟通协调（15%）
- 预防措施（10%）

**参考答案思路：**
1. 评估情况：
   - 确认布料规格和数量
   - 评估布料状态（是否受损）
   - 了解市场行情
2. 处理方案：
   - 寻找其他客户使用
   - 与供应商协商退货/换货
   - 降价销售给其他买家
   - 用于打样或其他用途
   - 财务处理（计提损失）
3. 沟通协调：
   - 与销售部门沟通
   - 与财务部门沟通
   - 与供应商协商
4. 预防措施：
   - 完善订单取消条款
   - 控制库存水平
   - 建立预警机制

## 四、跨部门协作类

### 情景 8：生产部门投诉

**背景：**
生产部门投诉你采购的坯布质量不稳定，影响了生产效率和产品质量。但你认为供应商的质量是合格的。你会怎么处理？

**考察点：** 沟通能力、问题分析、协调能力

**评分维度：**
- 倾听理解（20%）
- 问题调查（25%）
- 解决方案（25%）
- 沟通技巧（20%）
- 长期改善（10%）

**参考答案思路：**
1. 倾听和理解：
   - 了解生产部门的具体问题
   - 收集数据和证据
   - 表示重视和配合
2. 联合调查：
   - 到生产现场查看
   - 分析问题批次数据
   - 对比检验标准
3. 解决问题：
   - 如果是质量问题：与供应商沟通改进
   - 如果是标准问题：协调双方标准
   - 如果是使用问题：提供技术支持
4. 持续改善：
   - 建立定期沟通机制
   - 完善检验标准
   - 加强过程监控

### 情景 9：财务部门要求

**背景：**
财务部门要求你延长付款周期从 30 天到 60 天，以改善公司现金流。但你知道这可能影响供应商关系和价格。你会怎么做？

**考察点：** 平衡能力、沟通协调、策略思维

**评分维度：**
- 影响分析（25%）
- 沟通策略（25%）
- 解决方案（25%）
- 平衡能力（15%）
- 执行计划（10%）

**参考答案思路：**
1. 影响分析：
   - 评估对供应商关系的影响
   - 评估对价格的影响
   - 评估对交期的影响
2. 沟通协调：
   - 与财务部门沟通采购影响
   - 与供应商沟通可能性
   - 与领导汇报权衡
3. 解决方案：
   - 分供应商执行：战略供应商保持，一般供应商延长
   - 交换条件：延长账期换取价格优惠
   - 阶段实施：逐步延长，给供应商缓冲
4. 平衡策略：
   - 维护战略供应商关系
   - 选择议价能力强的供应商延长
   - 监控供应商表现

## 六、压力与挑战类

### 情景 12：多重任务

**背景：**
你同时面临以下任务：
1. 一个紧急订单需要 3 天内完成采购
2. 一份月度报告明天要交
3. 供应商明天要来拜访
4. 一批货物需要验收入库

你会如何安排？

**考察点：** 优先级管理、时间管理、抗压能力

**评分维度：**
- 优先级判断（30%）
- 时间安排（25%）
- 资源协调（20%）
- 沟通能力（15%）
- 应急处理（10%）

**参考答案思路：**
1. 优先级排序：
   - 紧急订单（P0）：影响生产
   - 货物验收（P0）：影响入库和付款
   - 月度报告（P1）：可以加班完成
   - 供应商拜访（P1）：可以调整时间
2. 时间安排：
   - 今天：处理紧急订单，安排验收
   - 晚上：加班完成报告
   - 明天：接待供应商，同步处理其他事务
3. 资源协调：
   - 请同事协助验收
   - 与领导沟通报告时间
   - 与供应商调整拜访安排

## 七、情景题答题技巧

### 7.1 STAR 法则

**S** - Situation（情景）：描述背景
**T** - Task（任务）：明确目标
**A** - Action（行动）：说明具体做法
**R** - Result（结果）：展示成果

**示例：**
```
S：去年我们面临原材料涨价 20% 的压力
T：需要在保证质量的前提下降低成本
A：我采取了集中采购、供应商整合、规格优化等措施
R：最终实现了成本降低 5%，同时质量保持稳定
```

### 7.2 结构化表达

```
第一步：分析问题
第二步：确定目标
第三步：制定方案
第四步：执行实施
第五步：评估效果
```

### 7.3 展示能力

- **专业能力**：展示专业知识
- **分析能力**：逻辑清晰，数据支撑
- **沟通能力**：表达清楚，有说服力
- **执行能力**：有具体行动计划
- **学习能力**：愿意学习新工具和方法

---

*提示：情景题没有标准答案，关键是展示你的思维方式、解决问题的能力和专业素养。结合实际经验，讲出具体案例会更有说服力。*
