# 第六章：提问的艺术（Prompt Engineering）

> Prompt = 你发给 AI 的那段话。写得好，AI 给出专业答案；写得差，AI 给出废话。

## 一、Prompt 的四要素

```
高质量 Prompt = 角色 + 任务 + 上下文 + 格式

示例：
角色：你是一位有 15 年经验的坯布采购专家
任务：请分析以下供应商报价的合理性
上下文：当前棉花价格 15,800 元/吨，历史均价 8.3 元/米
格式：用表格对比，给出排名和建议
```

### 角色设定
```
好的：你是一位纺织行业的采购总监
差的：帮我看看
→ 角色让 AI 从专业角度回答
```

### 任务明确
```
好的：请从价格、交期、付款三个维度对比5家供应商
差的：分析一下
→ 告诉 AI 具体做什么
```

### 上下文充分
```
好的：产品40s×40s 133×72，报价8.5元/米，历史价8.0-8.8，棉花15800元/吨
差的：这个价格合理吗？
→ AI 需要足够背景才能判断
```

### 格式指定
```
好的：用表格展示，包含供应商/单价/交期/评分/排名
差的：分析一下这些数据
→ 指定格式让输出更清晰
```

## 二、纺织采购常用模板

### 询价分析
```
你是坯布采购专家。请对比这3家供应商报价：
产品：纯棉坯布 40s×40s 133×72
供应商A：8.5元/米，30天交期，月结30天
供应商B：8.2元/米，45天交期，款到发货
供应商C：8.8元/米，20天交期，货到付款
棉花价格：15,800元/吨
请判断合理性、排名、建议、风险
```

### 合同审查
```
请审查这份合同[上传PDF]，重点：
1. 价格与报价是否一致
2. 质量标准是否明确
3. 违约责任是否对等
用风险等级（高/中/低）标注，给出修改建议
```

## 三、进阶技巧

### 迭代式提问
```
第一轮：初步分析
第二轮："请详细展开第3点"
第三轮："如果棉花涨10%呢？"
第四轮："整理成报告大纲"
```

### 多角色分析
```
请分别从采购经理、法务、财务三个角度分析这份合同
```

## 四、常见错误

| 错误 | 正确 |
|------|------|
| 太模糊："帮我看看" | 明确任务："对比5家供应商报价" |
| 没上下文 | 提供背景数据 |
| 一次性问太多 | 分步骤提问 |
| 不追问 | 迭代完善 |
| 直接采纳 | 重要决策仍需人工判断 |
