# 案例四：AI 合同智能审查

## 行业痛点

- 每月审查 10-20 份采购合同，每份 2-3 小时
- 容易遗漏关键条款（尤其是违约责任）
- 法务部门人手不足，排队等审核

## AI 审查实操

```
你：请审查这份坯布采购合同，重点检查：
1. 价格条款是否与报价单一致（报价：8.5元/米）
2. 质量标准是否明确
3. 交期条款是否合理
4. 违约责任是否对等
5. 是否有对我方不利的隐含条款

[上传合同 PDF]
```

### AI 审查报告

```
✅ 无问题：产品规格清晰、数量单位明确

⚠️ 需注意：
├── 第3.2条：交期写"约30天" → 改为"30个自然日内"
├── 第5.1条：质量标准引用"行业通用标准" → 改为具体国标 GB/T 406-2018
├── 第7.3条：违约金 0.1%/天 → 建议提高到 0.3%/天
└── 第8.1条：预付款 50% → 建议降低到 30%

❌ 存在风险：
├── 第9.2条：只约束买方违约，未约束卖方 → 增加供应商违约条款
├── 第11条：争议由供应商所在地管辖 → 改为买方所在地仲裁
└── 第12.3条：不可抗力定义过宽 → 收窄为自然灾害、战争等
```

## 效率对比

| 指标 | 人工审查 | AI 审查 | 提升 |
|------|---------|---------|------|
| 审查时间 | 2-3小时/份 | 10-15分钟/份 | 90%↓ |
| 遗漏率 | 10-20% | <5% | 75%↓ |

**⚠️ AI 审查不能替代法务，仅作为初筛工具**
