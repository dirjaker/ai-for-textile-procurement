# 案例二：AI 原料价格预测

## 行业痛点

| 原料 | 年价格波动 | 对采购的影响 |
|------|-----------|-------------|
| 棉花 | ±15-25% | 纯棉坯布成本波动 |
| 涤纶 | ±10-20% | 涤棉混纺成本波动 |
| 粘胶 | ±15-30% | 人棉坯布成本波动 |

## AI 如何预测价格？

```
AI 分析的数据维度：
├── 国际棉花期货（ICE）
├── 中国棉花价格指数（CC Index）
├── 储备棉轮出政策
├── 全球供需数据（USDA 月报）
├── 天气数据（影响产量）
├── 汇率数据（影响进口成本）
└── 历史价格走势（季节性规律）
```

## 实操步骤

### 第一步：收集数据

```
你：帮我整理近 3 年中国棉花价格指数的月度数据，分析季节性规律。

AI：历史规律：
- 3-5月：春季播种期，价格通常上涨 5-8%
- 9-11月：采摘季，价格通常下降 3-5%
- 12-2月：消费旺季，价格小幅上涨
```

### 第二步：预测分析

```
你：当前棉花价格 15,800 元/吨，近 6 个月上涨 8%，
    USDA 全球库存下降 5%，人民币贬值 2%。
    请预测未来 3 个月走势并给出采购建议。

AI：预测区间 15,200-16,500 元/吨（置信度 65%）
├── 短期：维持正常采购节奏
├── 中期：跌到 15,200 以下时增加库存 20%
└── 建议：与供应商签订 3 个月锁价合同
```

## 效率对比

| 指标 | 传统方式 | AI 辅助 | 提升 |
|------|---------|---------|------|
| 价格判断准确率 | 50-60% | 65-75% | +15% |
| 分析时间 | 每周4小时 | 每周30分钟 | 节省87% |
| 价格波动损失 | 5-10% | 2-5% | 降低50% |

## ⚠️ 重要提醒

1. AI 预测不是 100% 准确 — 只能作为决策参考
2. 黑天鹅事件无法预测
3. 结合多种信息源：AI 分析 + 行业专家 + 自身经验
