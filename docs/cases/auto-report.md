# 案例六：采购数据自动报表

## 行业痛点

采购经理每月花 1-2 天整理 Excel、做图表、写报告

## AI 自动化方案

```
你：分析这份采购数据[上传Excel]，生成月度报告，包括：
1. 采购概况（总金额、订单数、供应商数）
2. 按品类分析
3. 按供应商分析（TOP10）
4. 同比/环比
5. 成本异常预警
6. 下月建议

请生成 Word 格式，包含图表。
```

## 效率对比

| 报表类型 | 传统方式 | AI 辅助 | 节省 |
|---------|---------|---------|------|
| 周报 | 2小时 | 15分钟 | 87% |
| 月报 | 4小时 | 30分钟 | 87% |
| 季报 | 1天 | 1小时 | 87% |
| 年报 | 2-3天 | 2小时 | 90% |
