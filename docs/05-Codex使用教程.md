# 第五章：Codex 使用教程

> 面向非技术人员的 AI 助手使用指南

---

## 一、什么是 Codex Desktop？

### 1.1 简单介绍

**Codex Desktop** 是 OpenAI 推出的桌面应用程序，让你可以在电脑上直接使用 AI 助手。

**它能做什么：**
- 🗣️ **对话交流** - 像聊天一样和 AI 对话
- 📄 **处理文档** - 读取、分析、生成文档
- 📊 **分析数据** - 处理 Excel、CSV 等数据文件
- 🖼️ **理解图片** - 识别图片内容、提取文字
- 🔍 **搜索信息** - 联网搜索最新信息
- 💻 **执行代码** - 运行 Python 脚本处理任务

**类比：**
- ChatGPT = 网页版 AI 助手
- Codex Desktop = 电脑上的 AI 助手，功能更强

### 1.2 为什么选择 Codex Desktop？

| 对比项 | ChatGPT 网页版 | Codex Desktop |
|--------|---------------|---------------|
| **安装方式** | 打开浏览器访问 | 安装到电脑 |
| **文件处理** | 需要上传 | 直接读取本地文件 |
| **运行速度** | 受网络影响 | 本地运行更快 |
| **离线使用** | ❌ 不支持 | ⚠️ 部分支持 |
| **功能扩展** | 有限 | 支持插件 |

**适合你的场景：**
- ✅ 你经常需要处理 Excel 数据
- ✅ 你想让 AI 直接操作电脑上的文件
- ✅ 你需要批量处理多个文件
- ✅ 你希望操作更简单，不想学命令行

## 二、安装 Codex Desktop

### 2.1 系统要求

| 系统 | 要求 |
|------|------|
| **Windows** | Windows 10 或更高版本 |
| **macOS** | macOS 12.0 或更高版本 |
| **内存** | 8GB 以上推荐 |
| **存储空间** | 500MB 可用空间 |
| **网络** | 需要网络连接 |

### 2.2 下载安装

#### Windows 用户

**步骤 1：下载安装包**
- 访问 https://openai.com/codex
- 点击 "Download for Windows"
- 下载 `.exe` 安装包

**步骤 2：安装**
- 双击下载的安装包
- 按照提示点击 "下一步"
- 完成安装

**步骤 3：启动**
- 在桌面或开始菜单找到 "Codex" 图标
- 双击打开

#### macOS 用户

**步骤 1：下载安装包**
- 访问 https://openai.com/codex
- 点击 "Download for macOS"
- 下载 `.dmg` 安装包

**步骤 2：安装**
- 双击下载的 `.dmg` 文件
- 将 Codex 图标拖到 Applications 文件夹

**步骤 3：启动**
- 打开 Applications 文件夹
- 双击 "Codex" 图标

### 2.3 登录账号

**步骤 1：打开应用**
- 启动 Codex Desktop
- 点击 "Sign in with OpenAI"

**步骤 2：登录**
- 输入你的 OpenAI 账号和密码
- 如果没有账号，点击 "Sign up" 注册

**步骤 3：授权**
- 授权 Codex 访问你的账户
- 点击 "Allow"

**注意：**
- 需要有 OpenAI 账号
- 需要有可用的 API 额度
- 建议先充值 $5-10 测试

**主要区域：**
- **左侧**：历史记录和工具面板
- **中间**：对话区域
- **底部**：输入框和发送按钮

### 3.2 发送消息

**基本操作：**
1. 点击底部的输入框
2. 输入你想说的话
3. 点击 "发送" 按钮或按回车键

**示例对话：**
```
你：你好，帮我写一封询价邮件

AI：好的，请告诉我以下信息：
1. 产品名称和规格
2. 需要的数量
3. 期望的交货时间
4. 其他要求

你：纯棉坯布 40s×40s 133×72，需要 5000 米，30 天内交货

AI：好的，这是询价邮件：

尊敬的供应商：

您好！我司现需采购纯棉坯布，具体需求如下：
- 规格：40s×40s 133×72
- 数量：5000 米
- 交期：30 天内
- 质量要求：一等品

请提供报价，谢谢！

此致
敬礼
```

### 3.3 上传文件

**方式一：拖拽上传**
1. 打开文件所在的文件夹
2. 将文件拖到对话区域
3. 等待上传完成

**方式二：点击上传**
1. 点击输入框旁边的 📎 图标
2. 选择要上传的文件
3. 点击 "打开"

**支持的文件类型：**
- 📄 文档：PDF、Word、TXT、Markdown
- 📊 数据：Excel、CSV、JSON
- 🖼️ 图片：JPG、PNG、GIF、BMP
- 💻 代码：Python、JavaScript、HTML 等

## 五、采购工作实战

### 5.1 案例一：供应商报价分析

**场景：** 收到 5 家供应商的报价，需要对比分析

**操作步骤：**

```
第一步：整理报价数据
────────────────────────────────────────────────────────
你：我有 5 家供应商的报价，请帮我整理成表格并分析：

供应商 A：8.5 元/米，30 天交期，款到发货
供应商 B：8.2 元/米，45 天交期，月结 30 天
供应商 C：8.8 元/米，20 天交期，货到付款
供应商 D：8.0 元/米，60 天交期，月结 60 天
供应商 E：8.3 元/米，25 天交期，预付 30%

AI：好的，这是整理后的表格：

| 供应商 | 单价 | 交期 | 付款条件 |
|--------|------|------|----------|
| A | 8.5 | 30天 | 款到发货 |
| B | 8.2 | 45天 | 月结30天 |
| C | 8.8 | 20天 | 货到付款 |
| D | 8.0 | 60天 | 月结60天 |
| E | 8.3 | 25天 | 预付30% |

分析：
1. 价格最低：供应商 D（8.0 元/米）
2. 交期最短：供应商 C（20 天）
3. 付款最灵活：供应商 C（货到付款）

推荐：
- 如果急需：选择供应商 C（交期短、付款灵活）
- 如果追求低价：选择供应商 D（价格最低、账期长）
- 如果综合考虑：选择供应商 E（价格适中、交期短）
```

### 5.2 案例二：合同审查

**场景：** 收到供应商的合同，需要快速审查

**操作步骤：**

```
第一步：上传合同
────────────────────────────────────────────────────────
你：帮我审查这份合同，找出风险点
[上传合同 PDF]

AI：我来审查这份合同...

审查结果：

✅ 没有问题的条款：
- 产品规格明确
- 价格条款清晰

⚠️ 需要注意的条款：
1. 交货期：合同写"尽快交货"，建议明确具体天数
2. 付款方式：预付 50%，建议改为 30%
3. 质量标准：未明确验收标准，建议补充

❌ 存在风险的条款：
1. 违约责任：只约束买方，建议增加对卖方的约束
2. 争议解决：建议改为我方所在地法院

修改建议：
[列出具体修改建议]
```

## 六、使用技巧

### 6.1 提问技巧

**好的提问方式：**
```
明确任务 + 提供上下文 + 指定格式

示例：
"帮我分析这份供应商报价数据（上传文件），
找出性价比最高的 3 家供应商，
用表格形式展示，并说明推荐理由。"
```

**不好的提问方式：**
```
太模糊：帮我分析一下
没上下文：这个数据怎么样？
没格式：写个报告
```

### 6.2 文件处理技巧

**批量处理：**
```
你：帮我处理这个文件夹里的所有 Excel 文件
[选择文件夹]

AI：好的，我来批量处理...
[处理所有文件]
```

**指定格式：**
```
你：把分析结果保存为 PDF 格式

AI：好的，已保存为 PDF 文件：
[显示文件路径]
```

### 6.3 常见问题解决

**问题 1：AI 不理解我的需求**
```
解决：提供更多细节
- 说明背景
- 提供示例
- 指定具体要求
```

**问题 2：结果不准确**
```
解决：要求 AI 说明依据
- 让 AI 展示计算过程
- 要求 AI 说明信息来源
- 人工核实关键数据
```

**问题 3：文件无法打开**
```
解决：检查文件格式
- 确认文件类型支持
- 检查文件是否损坏
- 尝试转换格式后重新上传
```

## 八、常见问题

### Q1：Codex Desktop 和 ChatGPT 有什么区别？

**答：**
- ChatGPT 是网页版，功能有限
- Codex Desktop 是桌面版，功能更强
- Codex Desktop 可以直接操作本地文件
- Codex Desktop 支持更多插件

### Q2：需要付费吗？

**答：**
- 需要有 OpenAI 账号
- 需要有可用的 API 额度
- 按使用量计费
- 建议先充值 $5-10 测试

### Q3：支持哪些文件格式？

**答：**
- 文档：PDF、Word、TXT、Markdown
- 数据：Excel、CSV、JSON
- 图片：JPG、PNG、GIF、BMP
- 代码：Python、JavaScript、HTML 等

### Q4：如何保护数据安全？

**答：**
- 敏感数据先脱敏处理
- 不要上传包含密码、密钥的文件
- 使用后及时清理对话记录
- 重要数据建议本地处理

### Q5：网络不好怎么办？

**答：**
- 检查网络连接
- 尝试使用 VPN
- 减少上传大文件
- 使用离线功能（如有）
