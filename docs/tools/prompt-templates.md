# Prompt 模板库（纺织采购专用）

> **复制粘贴即可使用**，把 [方括号] 替换成你的实际信息

---

## 一、报价分析

### 模板1：供应商报价对比

```
你是一位有 15 年经验的纺织采购专家。请分析以下供应商报价：

产品规格：[40s×40s 133×72 纯棉坯布]

供应商报价：
1. [供应商A]：[8.5]元/米，交期[30]天，付款[月结30天]
2. [供应商B]：[8.2]元/米，交期[45]天，付款[款到发货]
3. [供应商C]：[8.8]元/米，交期[20]天，付款[货到付款]

当前棉花价格：[15,800]元/吨
历史采购均价：[8.3]元/米

请分析：报价合理性、综合性价比排名、采购建议、风险提示
```

### 模板2：价格合理性判断

```
请判断以下坯布报价是否合理：
产品：[纯棉坯布 60s×60s 90×88]
报价：[12.5]元/米
当前棉花价格：[16,000]元/吨
去年同期：[11.8]元/米

请分析涨幅是否合理、说辞是否可信、议价策略
```

## 二、合同审查

### 模板3：合同风险审查

```
请审查这份采购合同，找出风险点：
[上传合同 PDF]

审查重点：价格条款、质量标准、交期、违约责任、隐含不利条款
请用表格列出：| 条款 | 风险等级 | 问题 | 修改建议 |
```

## 三、供应商管理

### 模板4：供应商综合评估

```
请对以下供应商进行综合评估（质量40%/交付30%/价格20%/服务10%）：

供应商：[XX纺织]，合作[3]年，年采购额[¥500万]
来料合格率：[98.5%]，准时交货率：[95%]
价格偏差：[+3%]，客诉：[2次/年]，响应速度：[一般]

请给出：评分、等级(A/B/C/D)、优势风险、改善建议、采购策略
```

## 💡 进阶技巧

**记住背景：** 在对话开头说"我是纺织采购经理，主要采购纯棉和涤棉坯布"

**追问技巧：**
```
第一轮：初步分析
第二轮："请更详细地展开第3点"
第三轮："如果棉花涨10%，影响是多少？"
第四轮："整理成PPT大纲"
```
