# 30 分钟快速入门

> 只有 30 分钟？按这个操作，立刻体验 AI 的威力

## 第1步：注册（5分钟）

打开 https://chat.deepseek.com → 手机号注册 → 登录

## 第2步：首次对话（5分钟）

发送：
```
我是纺织采购经理，帮我写一封询价邮件。
产品：40s×40s 133×72纯棉坯布，数量5000米，30天内交货。
```

## 第3步：上传数据（10分钟）

上传一份采购 Excel → 发送"请分析采购金额TOP5的供应商"

## 第4步：合同审查（10分钟）

上传一份合同 PDF → 发送"请找出3个最大的风险点"
