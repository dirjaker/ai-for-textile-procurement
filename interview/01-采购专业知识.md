## 一、坯布基础知识

### 1.1 纱线与原料

**Q1：请解释纯棉纱线的规格表示方法（如 40s×40s）**

**考察点：** 基础知识

**答题要点：**
- 40s 表示纱线支数（英支）
- 支数越高，纱线越细
- 第一个 40s 是经纱，第二个 40s 是纬纱
- 高支纱用于高档面料，低支纱用于粗布

**Q2：纯棉、涤棉、人棉的主要区别是什么？各适用于什么产品？**

**考察点：** 材料特性

**答题要点：**

| 材料 | 特点 | 适用产品 |
|------|------|---------|
| 纯棉 | 吸湿透气、舒适、易皱 | 内衣、床品、婴童服装 |
| 涤棉 | 强度高、挺括、耐磨 | 工作服、衬衫、裤子 |
| 人棉 | 手感柔软、悬垂性好 | 裙子、里布、夏季服装 |

**Q3：什么是精梳棉和普梳棉？有什么区别？**

**考察点：** 工艺知识

**答题要点：**
- 普梳棉：普通梳理工艺，纤维排列不够整齐
- 精梳棉：增加精梳工序，去除短纤维和杂质
- 区别：精梳棉更光滑、强度更高、价格更贵
- 应用：精梳棉用于高档衬衫、内衣

**Q4：棉花的主要产地有哪些？不同产地的特点是什么？**

**考察点：** 原料知识

**答题要点：**
- 新疆棉：品质好、长度长、强度高
- 美棉：一致性好、杂质少
- 印度棉：价格便宜、质量参差不齐
- 非洲棉：长度长、适合高支纱

### 1.2 织造工艺

**Q5：平纹、斜纹、缎纹三种基本组织的特点和区别是什么？**

**考察点：** 织造知识

**答题要点：**

| 组织 | 特点 | 适用产品 |
|------|------|---------|
| 平纹 | 交织点多、结构稳定、耐磨 | 床单、工作服、帆布 |
| 斜纹 | 手感柔软、有光泽、厚实 | 牛仔布、卡其布、风衣 |
| 缎纹 | 光泽最好、手感最滑、不耐磨 | 旗袍、高档床品、里布 |

**Q6：什么是府绸？什么是纱卡？什么是线卡？**

**考察点：** 品种知识

**答题要点：**
- 府绸：平纹组织，高经密，有菱形颗粒效果，用于衬衫
- 纱卡：斜纹组织，纱线织造，耐磨，用于工作服
- 线卡：斜纹组织，股线织造，更挺括，用于裤料

**Q7：什么是提花面料？和平纹、斜纹有什么区别？**

**考察点：** 工艺知识

**答题要点：**
- 提花：通过组织变化形成花纹
- 区别：提花是织出来的花纹，不是印上去的
- 特点：花纹有立体感、不会脱落
- 应用：高档床品、窗帘、服装

**Q8：什么是色织面料？和印染面料有什么区别？**

**考察点：** 工艺知识

**答题要点：**
- 色织：先染纱线，再织布
- 印染：先织白布，再印花染色
- 区别：色织花纹更清晰、不会褪色，但成本更高
- 应用：色织用于格子布、条纹布；印染用于花布

### 1.3 质量指标

**Q9：坯布的主要质量指标有哪些？**

**考察点：** 质量知识

**答题要点：**

| 指标 | 说明 | 重要性 |
|------|------|--------|
| 密度 | 经密×纬密（根/英寸） | ⭐⭐⭐⭐⭐ |
| 幅宽 | 布面宽度（英寸或厘米） | ⭐⭐⭐⭐⭐ |
| 克重 | 每平方米重量（g/m²） | ⭐⭐⭐⭐ |
| 强力 | 断裂强力（N） | ⭐⭐⭐⭐ |
| 缩水率 | 洗涤后收缩比例 | ⭐⭐⭐⭐ |
| 色牢度 | 颜色牢固程度 | ⭐⭐⭐⭐ |
| 疵点 | 布面缺陷数量 | ⭐⭐⭐⭐ |

**Q10：什么是 AQL？在采购验收中如何应用？**

**考察点：** 质量检验

**答题要点：**
- AQL = Acceptable Quality Level，可接受质量水平
- 是一种抽样检验标准
- 根据批量大小确定抽样数量
- 规定允收的不良品数量
- 常用标准：AQL 2.5（一般检验）、AQL 4.0（宽松检验）

**Q11：常见的坯布疵点有哪些？如何识别？**

**考察点：** 质量识别

**答题要点：**

| 疵点 | 表现 | 原因 |
|------|------|------|
| 断经 | 经向有断裂 | 纱线强力不足或织造问题 |
| 断纬 | 纬向有断裂 | 纬纱问题或引纬故障 |
| 稀密路 | 经向有稀或密的条纹 | 送经或卷取不匀 |
| 纬斜 | 纬纱倾斜 | 织造张力不匀 |
| 破洞 | 布面有洞 | 机械损伤或纱线问题 |
| 油污 | 布面有油渍 | 设备漏油或操作不当 |

## 二、采购流程管理

**Q12：请描述完整的坯布采购流程**

**考察点：** 流程管理

**答题要点：**
```
需求确认 → 供应商选择 → 询价比价 → 合同签订 → 订单跟踪 → 验收入库 → 付款结算
```

**Q13：采购计划如何制定？需要考虑哪些因素？**

**考察点：** 计划能力

**答题要点：**
- 需求来源：销售预测、生产计划、安全库存
- 考虑因素：
  - 历史采购数据
  - 季节性因素
  - 供应商交期
  - 价格走势
  - 资金预算
- 计划周期：月度计划、季度计划、年度计划

**Q14：如何进行采购成本分析？**

**考察点：** 成本管理

**答题要点：**
- 成本构成：原料成本、加工成本、运输成本、仓储成本、资金成本
- 分析方法：
  - 成本分解：逐项分析
  - 对标分析：与市场对比
  - 趋势分析：历史变化
  - 供应商对比：不同供应商对比

**Q15：如何制定采购预算？**

**考察点：** 预算管理

**答题要点：**
- 数据基础：历史采购数据、需求预测
- 预算方法：
  - 零基预算：从零开始估算
  - 增量预算：在上年基础上调整
  - 滚动预算：按季度滚动调整
- 预算控制：定期对比、偏差分析、调整优化

## 三、供应商管理

**Q16：如何开发新的坯布供应商？**

**考察点：** 供应商开发

**答题要点：**
- 信息来源：行业展会、同行推荐、网络搜索、供应商自荐
- 筛选标准：资质、规模、产品范围、质量体系、价格
- 评估流程：
  1. 初步筛选（资质审核）
  2. 样品评估（质量测试）
  3. 现场审核（生产能力、管理水平）
  4. 小批量试用（验证实际表现）
  5. 正式合作（签订协议）

**Q17：如何评估供应商的综合能力？**

**考察点：** 供应商评估

**答题要点：**

| 维度 | 评估内容 | 权重建议 |
|------|---------|---------|
| 质量 | 质量体系、合格率、投诉处理 | 30% |
| 价格 | 价格水平、付款条件、降本能力 | 25% |
| 交期 | 准时率、交期稳定性、应急能力 | 20% |
| 服务 | 响应速度、技术支持、售后 | 15% |
| 技术 | 研发能力、创新性、设备水平 | 10% |

**Q18：供应商出现问题时如何处理？**

**考察点：** 问题处理

**答题要点：**
- 质量问题：
  1. 暂停收货，隔离问题产品
  2. 与供应商沟通，要求原因分析
  3. 要求整改方案和预防措施
  4. 加强后续检验
  5. 评估是否继续合作

- 交期问题：
  1. 了解延误原因
  2. 协商补救措施（加急、分批）
  3. 评估损失和索赔
  4. 调整后续订单计划

**Q19：如何维护与战略供应商的关系？**

**考察点：** 供应商关系管理

**答题要点：**
- 建立互信：信息透明、承诺兑现
- 定期沟通：高层会晤、季度回顾
- 合作共赢：联合改进、共享收益
- 长期承诺：签订长期协议、稳定订单
- 风险共担：共同应对市场变化

## 四、质量控制

**Q20：坯布验收的标准和流程是什么？**

**考察点：** 质量验收

**答题要点：**
- 验收依据：
  - 合同约定的技术指标
  - 国家/行业标准（GB/T 标准）
  - 封样样品
- 验收流程：
  1. 外观检验：布面疵点、颜色、手感
  2. 规格检验：密度、幅宽、克重
  3. 内在质量：强力、缩水率、色牢度
  4. 抽样检验：AQL 标准
  5. 判定结论：合格/不合格/让步接收

**Q21：如何处理质量争议？**

**考察点：** 争议处理

**答题要点：**
- 保留证据：问题样品、检验记录、照片
- 第三方检测：送专业检测机构
- 协商解决：双方认可的检测结果
- 依据合同：按照合同条款执行
- 记录存档：为后续合作提供参考

**Q22：如何建立质量预防机制？**

**考察点：** 质量管理

**答题要点：**
- 事前控制：
  - 明确质量标准
  - 供应商能力评估
  - 样品确认
- 事中控制：
  - 生产过程监控
  - 中期检验
  - 问题及时反馈
- 事后控制：
  - 成品检验
  - 问题分析改进
  - 供应商辅导

## 五、成本控制

**Q23：有哪些降低采购成本的方法？**

**考察点：** 成本优化

**答题要点：**
- 集中采购：合并需求，提高议价能力
- 长期协议：签订年度协议，锁定价格
- 供应商整合：减少供应商数量，增加单个供应商份额
- 付款条件：利用账期，降低资金成本
- 替代材料：使用性价比更高的材料
- 规格优化：避免过度规格化
- 批量采购：利用批量折扣
- 竞争性谈判：多家比价，择优选择

**Q24：如何进行有效的采购谈判？**

**考察点：** 谈判技巧

**答题要点：**
- 准备工作：
  - 了解市场行情
  - 分析供应商成本结构
  - 明确谈判目标（理想、底线、替代方案）
  - 准备谈判策略
- 谈判技巧：
  - 倾听：了解供应商的真实想法
  - 提问：获取更多信息
  - 让步：有条件地让步
  - 打包：多个议题一起谈
- 注意事项：
  - 保持专业态度
  - 追求双赢
  - 记录谈判结果

**Q25：如何计算采购总拥有成本（TCO）？**

**考察点：** 成本分析

**答题要点：**
- TCO = 采购价格 + 交易成本 + 使用成本 + 处置成本
- 包括：
  - 采购价格：货款
  - 交易成本：运输、检验、谈判成本
  - 使用成本：质量损失、加工损耗
  - 处置成本：退货、索赔成本
- 应用：比较不同供应商的真实成本

## 六、供应链管理

**Q26：如何应对供应链中断风险？**

**考察点：** 风险管理

**答题要点：**
- 风险识别：供应商单一、地域集中、原料依赖
- 预防措施：
  - 多元化供应商
  - 安全库存
  - 备份供应商
- 应急方案：
  - 快速切换供应商
  - 调整生产计划
  - 客户沟通

**Q27：什么是 VMI？在什么情况下适用？**

**考察点：** 供应链创新

**答题要点：**
- VMI = Vendor Managed Inventory，供应商管理库存
- 概念：供应商管理采购方的库存
- 适用条件：
  - 需求稳定
  - 供应商可靠
  - 信息系统对接
- 优势：降低库存成本、提高响应速度
- 风险：库存准确性、供应稳定性

**Q28：如何优化采购交期？**

**考察点：** 交期管理

**答题要点：**
- 需求端：
  - 准确预测需求
  - 提前下达订单
  - 减少需求变动
- 供应端：
  - 选择交期可靠的供应商
  - 建立安全库存
  - 签订交期保障条款
- 过程管理：
  - 订单跟踪
  - 进度预警
  - 及时沟通

## 七、行业知识

**Q29：当前棉纺行业的主要挑战是什么？**

**考察点：** 行业认知

**答题要点：**
- 原料价格波动：棉花价格受政策、天气、国际市场影响
- 环保要求：节能减排、绿色生产
- 劳动力成本：用工成本上升、招工难
- 市场竞争：东南亚低成本竞争
- 技术升级：智能制造、数字化转型

**Q30：棉纺行业的发展趋势是什么？**

**考察点：** 趋势判断

**答题要点：**
- 功能化：功能性面料需求增加
- 绿色化：环保、可持续发展
- 智能化：智能制造、数字化采购
- 品牌化：从代工向品牌转型
- 差异化：高端化、个性化

**Q31：如何看待纺织行业的数字化转型？**

**考察点：** 创新思维

**答题要点：**
- 趋势必然：提高效率、降低成本
- 应用领域：
  - 数字化采购：在线询价、电子合同
  - 智能制造：自动化生产、质量监控
  - 供应链协同：信息共享、快速响应
- 个人准备：学习新工具、提升数字素养

## 八、法规与标准

**Q32：坯布采购需要关注哪些法规和标准？**

**考察点：** 法规知识

**答题要点：**
- 产品标准：
  - GB/T 406-2018《棉本色布》
  - GB/T 5325《精梳涤棉混纺本色布》
  - FZ/T 10004《棉及化纤纯纺、混纺本色布检验规则》
- 安全标准：
  - GB 18401《国家纺织产品基本安全技术规范》
  - 甲醛含量、pH 值、色牢度等
- 环保标准：
  - Oeko-Tex Standard 100
  - GOTS（全球有机纺织品标准）

**Q33：什么是 Oeko-Tex 认证？为什么重要？**

**考察点：** 认证知识

**答题要点：**
- Oeko-Tex 是国际纺织品生态认证
- 测试有害物质含量
- 分为多个级别（婴儿、直接接触皮肤等）
- 重要性：
  - 出口欧美市场的准入要求
  - 消费者健康保障
  - 品牌形象提升
